## 🧠 About the Project
CampusCare+ is a privacy-first platform that allows students to report campus issues anonymously, ensuring safety, transparency, and accountability.

---

## ✨ Features
- 🔒 Anonymous Reporting  
- 📊 Admin Dashboard  
- 🔄 Status Tracking  
- 📈 Analytics & Insights  
- 🔔 Real-Time Notifications  
- 🛡️ Spam Protection
